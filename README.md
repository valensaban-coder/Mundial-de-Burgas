# 🍔 Mundial de Burgas

App para puntuar hamburgueserías entre amigos. Cargás el local, cada jurado le pone su nota y se arma un ranking automático compartido para todos.

---

## ¿Cómo funciona?

**Pestaña Agregar**
- Ingresás el nombre de la hamburguesería
- Cada jurado (Sira, Agusto, Valen y Male) pone su puntaje del 1 al 10 — con decimales y coma (ej: 8,50)
- Opcionalmente cada uno escribe una reseña
- Se calcula el promedio automáticamente antes de confirmar

**Pestaña Tabla**
- Ranking ordenado de mayor a menor puntaje
- Se muestra el puntaje individual de cada jurado y el promedio final
- Las reseñas se pueden expandir tocando el nombre del jurado
- Podés editar o borrar cualquier entrada

**Sincronización**
- Los datos se guardan online y son compartidos en tiempo real
- Cualquiera que entre a la URL ve la misma tabla, sin importar desde qué dispositivo
